# NexusHR

NexusHR is a Human Resource Management System (HRMS) developed using Spring Boot and React.

## Tech Stack

### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- MySQL
- JPA / Hibernate

### Frontend
- React
- Bootstrap
- Axios
- React Router

## Features

- User Authentication
- Employee Management
- Attendance Management
- Leave Management
- Payroll
- Performance Reviews
- Notifications
- Announcements
- PDF Export
- Role-Based Access

## Roles

- Admin
- HR
- Manager
- Employee
